# heacoll

**hea**ders **coll**ector

Take as input on stdin a list of urls and print on stdout all the unique headers found. 

### Install

- `git clone https://github.com/edoardottt/lit-bb-hack-tools`
- `cd lit-bb-hack-tools/heacoll`
- `go build -o heacoll`
- `sudo cp heacoll /usr/bin`

### Usage

`cat urls | heacoll`
